WAV Reverse Engineering Tool

Analyze audio tracks for tempo, key, chords, notes, instruments, effects, and stems. This site documents the project and how to run the interactive Streamlit app locally.

Quick Start (CLI)

Install dependencies and analyze a file from the command line.

pip install -r wav_reverse_engineer/requirements.txt
python wav_reverse_engineer/cli.py analyze path/to/track.wav -o output --effects --instruments --separate hpss --export-json

Run the Web App

Launch the Streamlit UI for a user-friendly experience.

pip install -r wav_reverse_engineer/requirements.txt
# Optional heavy extras: panns-inference, torch, spleeter
pip install -r wav_reverse_engineer/requirements-optional.txt
streamlit run wav_reverse_engineer/webapp/app.py

Then upload a WAV/MP3 and explore analysis, effects, instruments, and stems.

Features

  • Visualization: waveform, spectrogram, mel, chroma, beat/pitch tracking
  • Tempo, key/mode, MFCCs, spectral features
  • Chord detection and progression summarization
  • Note onsets and pitch (YIN)
  • Advanced effects: RT60, spectral tilt, THD, compression index, loudness
  • Instrument recognition (PANNs if available; heuristic fallback)
  • Source separation: HPSS (built-in) or Spleeter (2/4/5 stems)
